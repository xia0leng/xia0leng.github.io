# <a href="#吡拉西坦" class="toc-anchor"></a> 吡拉西坦

**吡拉西坦(piracetam)**
俗称脑复康，是脑功能改善药、抗脑血管病药、抗中毒性脑病药、抗机械脑损伤药、抗镰状细胞性贫血药。

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理尚不完全清楚1" class="toc-anchor"></a> 作用机理<sub>（尚不完全清楚）</sub><sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 正变构受体：AMPA
- 提升功能：NMDA
- 提升功能：毒蕈碱乙酰胆碱/mAChr
- 促进合成：乙酰胆碱
- 促进合成：脑内ATP
- 增加大脑部分的血流量、耗氧量

#### <a href="#药物作用2有凑字数嫌疑" class="toc-anchor"></a> 药物作用<sup><a href="#fn2" id="fnref2">[2]</a></sup><sub>【有凑字数嫌疑】</sub>

- **主要的副作用：睡眠障碍**
- **急性的副作用：肌肉痉挛、震颤**
- 主要的主作用：运动机能亢进、认知感知强化、记忆力强化、梦境强化、性欲强化、<a href="/Effects/Antidissociation"
  class="is-internal-link is-valid-page">反解离</a>、反<a href="/Effects/Delirium"
  class="is-internal-link is-valid-page">谵妄</a>、焦虑抑制、抗ADHD、抗镰状细胞性贫血

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

## <a href="#药物使用" class="toc-anchor"></a> 药物使用

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

　　华意：瓶装0.4g×1瓶×每瓶100片，价格约为18元\
　　人福医药：瓶装0.4g×1瓶×每瓶100片，价格约为19元\
　　东北制药：瓶装0.4g×1瓶×每瓶100片，价格约为19元

#### <a href="#剂量-~以60kg无耐药性为标准3" class="toc-anchor"></a> 剂量 ~【以60kg，无耐药性为标准】<sup><a href="#fn3" id="fnref3">[3]</a></sup>

- 医学剂量：口服每次2t ~ 4t，每日3次，4 ~
  8周为一疗程。儿童量减半<sup><a href="#fn4" id="fnref4">[4]</a></sup>
- 起效剂量：0.5t~1t
- 轻度剂量：1t~5t
- 中度剂量：5t~7t
- 高度剂量：7t~8t
- 重度剂量：8t~12.5t
- 死亡剂量：12.5t+
- 推荐剂量：3t
- 最高剂量：8t

**期间：** <sub>【以距离服药已过去的时间为标准】</sub>\
　　起效时：30min~90min\
　　顶峰时：3h~4h\
　　总效时：2d

> 需注意，体重不同剂量不同，上述剂量是针对60kg的人的。你的体重除以60，再乘以上述之中的剂量，即为适应你的剂量

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

吡拉西坦对缺氧的逆行性遗忘有改善，例如亚硝酸钠导致的缺氧<sup><a href="#fn5" id="fnref5">[5]</a></sup>

吡拉西坦的作用机理尚不清楚，但可见的是，吡拉西坦算作一种脏药

关于半数致死量。大鼠为5.6g/kg，小鼠为20g/kg，可见急性毒性极低。作为对比，在大鼠中，维生素C为12g/kg，食盐为3g/kg<sup><a href="#fn6" id="fnref6">[6]</a></sup>

吡拉西坦可以造成头痛，然而这病因应该是合成乙酰胆碱时缺少原料造成的。于此，你应该使用胞磷胆碱钠等可以补充胆碱的药物<sup><a href="#fn7" id="fnref7">[7]</a></sup>，来反制这种头痛

吡拉西坦的药片较大，味道较苦，但没有右美沙芬苦

据一则很久以前的报告，吡拉西坦可以反制优米造成的晕迫
