# <a href="#苯海拉明" class="toc-anchor"></a> 苯海拉明

> 苯海拉明基本不能用于od，且作为谵妄剂具有高毒性

**苯海拉明(Diphenhydramine, DPH)**
是第一代抗过敏药、抗晕动病药，属于谵妄剂。一般以盐酸配合物，即**盐酸苯海拉明**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理-1" class="toc-anchor"></a> 作用机理 <sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 反激受体：组胺/H<sub>1</sub>
- 拮抗受体：毒蕈碱乙酰胆碱/mAChR/M<sub>1</sub>, M<sub>2</sub>,
  M<sub>3</sub>, M<sub>4</sub>, M<sub>5</sub>
- 阻滞通道：Na<sup>+</sup>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：<a href="/Effects/Delirium"
  class="is-internal-link is-valid-page">谵妄</a>、精神病状、记忆力丧失、阿兹海默症老年痴呆、死亡、抗胆碱能综合征、呼吸抑制**
- 主要的主作用：镇静

#### <a href="#药代动力学2" class="toc-anchor"></a> 药代动力学<sup><a href="#fn2" id="fnref2">[2]</a></sup>

苯海拉明的口服生物利用度为40%至60%。约2h~3h，血药浓度达到峰值。于体内经两次代谢，主要途径是叔胺的两次连续去甲基化，通俗点说就是拔两根草

罄竹难书的半衰期

苯海拉明的消除半衰期尚未完全阐明，但在健康成人中似乎在2.4h~9.3h之间

1985年抗组胺药药代动力学综述发现，五项研究中苯海拉明的消除半衰期在3.4h~9.3h之间，中位消除半衰期为4.3h

随后1990年的研究发现，苯海拉明的消除半衰期在儿童中为5.4h，在年轻人中为9.2h，在老年人中为13.5h

1998年的一项研究发现，年轻男性的半衰期为4.1h±0.3h，老年男性为7.4h±3.0h，年轻女性为4.4h±0.3h，老年女性为4.9h±0.6h

在 2018 年针对儿童和青少年的一项研究中，苯海拉明的半衰期为8h~9h

一般取5h就好啦

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

苯海拉明引起记忆力废退、谵妄、阿兹海默症，原因是抗胆碱能

据称，苯海拉明与右美沙芬联用具有特殊的加成？报告表明，加成并不明显

苯海拉明是美国最常被滥用的非处方药之一<sup><a href="#fn5" id="fnref5">[5]</a></sup>
