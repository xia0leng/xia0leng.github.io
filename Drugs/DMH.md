# <a href="#茶苯海明" class="toc-anchor"></a> 茶苯海明

> 茶苯海明基本不能用于od，且作为谵妄剂具有高毒性

**茶苯海明(Dimenhydrinate, DMH)**
是抗晕动病药、抗放射病药，属于谵妄剂，为复方药物。茶苯海明本身整合了8-氨茶碱，即<a href="/zh/Drugs/DPH"
class="is-internal-link is-valid-page">苯海拉明</a>8-氨茶碱。**基本不能od**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理-1" class="toc-anchor"></a> 作用机理 <sup><a href="#fn1" id="fnref1">[1]</a></sup>

<a href="/zh/Drugs/DPH"
class="is-internal-link is-valid-page">苯海拉明</a>部分——

- 反激受体：组胺/H<sub>1</sub>
- 拮抗受体：毒蕈碱乙酰胆碱/mAChR/M<sub>1</sub>, M<sub>2</sub>,
  M<sub>3</sub>, M<sub>4</sub>, M<sub>5</sub>
- 阻滞通道：Na<sup>+</sup>

氯茶碱部分——

- 提供兴奋，反制由苯海拉明带来的嗜睡副作用

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：<a href="/Effects/Delirium"
  class="is-internal-link is-valid-page">谵妄</a>、精神病状、记忆力丧失、阿兹海默症老年痴呆、死亡、抗胆碱能综合征、呼吸抑制**
- 主要的主作用：镇静

#### <a href="#药代动力学2" class="toc-anchor"></a> 药代动力学<sup><a href="#fn2" id="fnref2">[2]</a></sup>

茶苯海明被口服或舌下含服后，约2小时，血药浓度达到峰值。半衰期为5~6h

起效时：30min~1.5h\
来效时：45min~1.5h\
顶峰时：1h~4h\
退效时：2h~6h\
残效时：24h+\
总效时：3h~10h

#### <a href="#一些事项" class="toc-anchor"></a> 一些事项

茶苯海明引起记忆力废退、谵妄、阿兹海默症，原因是抗胆碱能

我有点私念，尽管茶苯海明算是个复方药物，但由于其被整合太久了所以单独做个页面也没事对吧

不建议和右美一起吃！吃了之后幻觉直接烟消云散，还会导致脱水，同时增强兴奋感。
