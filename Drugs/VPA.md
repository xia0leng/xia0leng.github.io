# <a href="#丙戊酸" class="toc-anchor"></a> 丙戊酸

**丙戊酸(Valproate, VPA)** 是抗癫痫药、抗躁狂药，属于镇静剂

通常以盐形式而成药剂，最常为**丙戊酸钠(Sodium valproate, VPS)** 与
**丙戊酸镁(Magnesium valproate, VPM)**
。丙戊酸钠倾向于抗癫痫（癫痫与钠有关），丙戊酸镁倾向于抗躁狂（镁有心境稳定作用）。**无法od**\
<img src="/vp.svg" class="align-right" alt="vp.svg" />\
实际上，丙戊酸盐的主要药效来自于丙戊酸，其全称为2-丙基戊酸，如图，结构就像一个鱼尾巴

## <a href="#药理作用1" class="toc-anchor"></a> 药理作用<sup><a href="#fn1" id="fnref1">[1]</a></sup>

#### <a href="#作用机理尚不明确" class="toc-anchor"></a> 作用机理【尚不明确】

阻断通道：Na<sup>+</sup>（电压门控）\
影响通道：Ca<sup>2+</sup>\
提高水平：GABA\
抑制合成与释放：一些神经物质

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

**主要的副作用：恶心、疲困、呕吐、畏食、腹泻**\
**少见的副作用：嗜睡、震颤、共济失调、脱发、血小板异常**\
主要的主作用：抗癫痫、抗躁狂、心境稳定

#### <a href="#药代动力学2" class="toc-anchor"></a> 药代动力学<sup><a href="#fn2" id="fnref2">[2]</a></sup>

丙戊酸口服后迅速且几乎完全从肠道吸收，并绝大多数于肝脏代谢，代谢物繁多，已知超过20种。半衰期为(16±3)h，与酶诱导剂联合使用时可缩短至4h~9h

#### <a href="#剂量-以60kg为标准" class="toc-anchor"></a> 剂量 <sub>【以60kg为标准】</sub>

丙戊酸镁医用剂量：每次0.2g，逐渐增加到每次0.3g ~ 0.4g。每日2 ~
3次，最高剂量不可以超过每日1.6g

丙戊酸钠医用剂量：每日3次，每次15mg/kg，即每次0.9g

#### <a href="#期间-以距离服药已过去的时间为标准缺少数据"
class="toc-anchor"></a> 期间 <sub>【以距离服药已过去的时间为标准】【缺少数据】</sub>

药效持续时间上，丙戊酸镁片为6h~8h，丙戊酸镁缓释片为12小时到24小时

药效持续时间上，丙戊酸钠片为3h~4h，丙戊酸钠缓释片为12小时到24小时
