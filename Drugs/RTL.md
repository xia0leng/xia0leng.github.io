# <a href="#哌醋甲酯" class="toc-anchor"></a> 哌醋甲酯

> 冰的相似物，第一类精神药物，同理<a href="/Drugs/MOP" class="is-internal-link is-valid-page">吗啡</a>——非医用就是毒品\
> **不要转卖或邮寄。转卖算贩毒，邮寄算运毒**

**哌醋甲酯(Methylphenidate)**
又名**哌甲酯**，商品名**利他林(Ritalin)**，是抗ADHD药，属于苯丙胺类兴奋剂/**冰的相似物**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理1" class="toc-anchor"></a> 作用机理<sup><a href="#fn1" id="fnref1">[1]</a></sup>

抑制再摄取：多巴胺（中枢神经系统中）\
抑制再摄取：正肾素\
激动受体：多巴胺、肾上腺素/D<sub>1</sub>,
α<sub>2</sub>（间接于前额皮质中）\
激动受体：血清素/5-HT<sub>1A</sub>, 5-HT<sub>2B</sub>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用**：成瘾、躁狂、降智（退效后）、高血压<sup><a href="#fn2" id="fnref2">[2]</a></sup>
- 主要的主作用：兴奋、抗ADHD、促智

#### <a href="#药代动力学3" class="toc-anchor"></a> 药代动力学<sup><a href="#fn3" id="fnref3">[3]</a></sup>

口服哌醋甲酯的生物利用度为11% ~
52%，且右哌醋甲酯比左旋醋哌甲酯高得多，不太相关于进食状态。作用持续时间可达2h
~ 8h，半衰期为2~3 小时，具体取决于个人。约2h，血压浓度达峰

哌醋甲酯在肝脏中被CES1A1酶代谢为利他林酸，并有97%处于代谢物以肾排泄，1%~3%处于代谢物以粪便排泄。不到1%的药物处于原形以肾排泄

## <a href="#药物使用" class="toc-anchor"></a> 药物使用

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

盐酸哌甲酯缓释片——【专注达】(Concerta)\
　　西安杨森：瓶装18mg/27mg/36mg/54mg/72mg×16片，价格不明

#### <a href="#剂量-以60kg无耐药性为标准4" class="toc-anchor"></a> 剂量 <sub>【以60kg，无耐药性为标准】</sub><sup><a href="#fn4" id="fnref4">[4]</a></sup>

医用剂量：一天一颗\
起效剂量：5mg\
剩下的不敢抄了

#### <a href="#期间5" class="toc-anchor"></a> 期间<sup><a href="#fn5" id="fnref5">[5]</a></sup>

起效时：20min~1h\
来效时：20min~1h\
顶峰时：1.5h~2.5h\
退效时：12h\
总效时：4h~6h

## <a href="#一些事项" class="toc-anchor"></a> 一些事项
