# <a href="#加巴喷丁" class="toc-anchor"></a> 加巴喷丁

**加巴喷丁(Gabapentin, PR)**
是抗癫痫药、神经止痛药、抗惊厥药，属于去焦虑剂。加巴喷丁本身就是一种酸，即1-(氨甲基)环己烷乙酸。滑雪衫

就像<a href="/Drugs/CDI" class="is-internal-link is-valid-page">可待因</a>是青春版的<a href="/Drugs/MOP" class="is-internal-link is-valid-page">吗啡</a>，同理，加巴喷丁也是青春版的<a href="/Drugs/PR" class="is-internal-link is-valid-page">普瑞巴林</a>。然而这两者的药代动力学数据却非常不同，且由于加巴喷丁效力较弱，因此加巴喷丁主要针对于12岁以下的患者。明打石锤的普瑞巴林儿童版了哈

## <a href="#药理作用1" class="toc-anchor"></a> 药理作用<sup><a href="#fn1" id="fnref1">[1]</a></sup>

#### <a href="#作用机理" class="toc-anchor"></a> 作用机理

- 阻断通道：Ca<sup>2+</sup>（电压依赖性α~2δ亚基）（由此引发以下作用）
- 抑制释放：乙酰胆碱/ACh
- 抑制释放：谷氨酸/Glu
- 抑制释放：P物质/SP/Substance P
- 抑制释放：正肾素/NE
- 间接提高水平：GABA

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：晕迫、记忆力衰退、认知降低、焦虑、阿兹海默症**<sup><a href="#fn2" id="fnref2">[2]</a></sup>
- 主要的主作用：社交能力提升、去除焦虑、<a href="/Effects/Dissociation"
  class="is-internal-link is-valid-page">解离</a> 、镇静

#### <a href="#药代动力学3" class="toc-anchor"></a> 药代动力学<sup><a href="#fn3" id="fnref3">[3]</a></sup>

加巴喷丁在体内几乎不代谢，并处于原形以肾排泄

较高剂量下，加巴喷丁的生物利用度降低且峰值水平延迟。对于剂量与利用度的关系

- 100mg时约为80% —— 80mg
- 300mg时约为60% —— 180mg
- 400mg时约为47% —— 188mg
- 800mg时约为34% —— 272mg
- 1200mg时约为33% —— 396mg
- 1600mg时约为27% —— 432mg
- 若联合口服吗啡，0.6g时口服生物利用度增加50%

不同于<a href="/Drugs/PR" class="is-internal-link is-valid-page">普瑞巴林</a>，加巴喷丁不需要空腹。但应当避免胃内酸性过高，例如避免饮用可乐。<sup><a href="#fn4" id="fnref4">[4]</a></sup>关于血药浓度达峰时，0.1g时约1.7h，高剂量时增加至3h~4h

加巴喷丁的消除半衰期为5h~7h

同理普瑞巴林，加巴喷丁并不能直接调控GABA水平

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

　　迭力：盒装0.1g×每盒48颗，价格约6元\
　　派汀：盒装0.3g×每盒10颗，价格约6元

#### <a href="#剂量-以60kg03g为1t无耐药性为标准5" class="toc-anchor"></a> 剂量 <sub>【以60kg，0.3g为1t，无耐药性为标准】</sub><sup><a href="#fn5" id="fnref5">[5]</a></sup>

医学剂量：涉及儿童用药，请遵医嘱\
起效剂量：1t\
轻度剂量：1t~3t\
中度剂量：3t~5t\
高度剂量：5t~8t\
重度剂量：8t+\
死亡剂量：【缺少数据】\
推荐剂量：4t~5t\
最高剂量：8t

#### <a href="#期间-以距离服药已过去的时间为标准" class="toc-anchor"></a> 期间 <sub>【以距离服药已过去的时间为标准】</sub>

起效时：30min~2h\
顶峰时：2h~3h\
总效时：6h~10h
