# <a href="#奥氮平" class="toc-anchor"></a> 奥氮平

> **比他妈<a href="/zh/Drugs/QTP"
> class="is-internal-link is-valid-page">喹硫平</a>还他妈罄竹难书的毒药！去他妈的奥氮平！！！**

**奥氮平(Olanzapine, OZP)**
是一种旧年的非典型抗精神病药，属于多种神经递质受体拮抗剂，目前已逐渐弃用。化学上是二苯并噻氮卓类衍生物。**无法滥用**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

> **罄竹难书**

#### <a href="#作用机理1你没看错就是这么多" class="toc-anchor"></a> 作用机理<sup><a href="#fn1" id="fnref1">[1]</a></sup><sub>你没看错，就是这么多！</sub>

- 拮抗受体：α<sub>1A</sub>, α<sub>1B</sub>, α<sub>2A</sub>,
  α<sub>2B</sub>, α<sub>2C</sub>
- 拮抗受体：多巴胺/DA<sub>1</sub>, DA<sub>2</sub>, DA<sub>2L</sub>,
  DA<sub>2S</sub>, DA<sub>3</sub>, DA<sub>4</sub>, DA<sub>4.2</sub>,
  DA<sub>4.4</sub>, DA<sub>5</sub>,
- 拮抗受体：血清素/5-HT<sub>1A</sub>, 5-HT<sub>3</sub>,
  5-HT<sub>6</sub>, 5-HT<sub>7</sub>
- 反向激动：血清素/5-HT<sub>2A</sub>, 5-HT<sub>2B</sub>,
  5-HT<sub>2C</sub>
- 拮抗受体：组胺/H<sub>2</sub>, H<sub>3</sub>, H<sub>4</sub>,
- 反向激动：组胺/H<sub>1</sub>
- 拮抗受体：毒蕈碱乙酰胆碱/mAChR/M<sub>1</sub>, M<sub>2</sub>,
  M<sub>3</sub>, M<sub>4</sub>, M<sub>5</sub>
- 阻断通道：hERG
- 完全激动：5-HT<sub>5A</sub>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：抑郁、增肥增重、谵妄、抗胆碱能综合征、阿兹海默症、认知衰退**
- 主要的主作用：抗精神分裂、反幻觉、镇静

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

奥氮平主要通过CYP1D6同工酶代谢，另外CYP2D6酶也参与其代谢。<sup><a href="#fn2" id="fnref2">[2]</a></sup>半衰期与达峰时间不明。
