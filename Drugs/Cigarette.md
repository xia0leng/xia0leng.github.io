## <a href="#药物效果" class="toc-anchor"></a> **药物效果：**

**上面所有有害物质共同作用下，造成的效果有：**

**即时"正面"效果：**

使人放松，镇静，愉悦，兴奋，缓解焦虑，抗疲劳

**即时负面效果：**

恶心，食欲不振，无力感，难以思考等

**长期效果：**

**心理：**生理性上瘾，致郁，长期兴奋，失眠，易怒，降智

**生理：**诱发心脑血管疾病，损害口腔、肺部及内脏健康，头痛，减少寿命，致癌，男女皆有的生育及性功能影响

**外表：**改变皮肤质量和面相，消瘦

**自身感受：**食欲不振，茶饭不思，躁动，难以集中精力，学习能力下降，

**其他：**口腔，手指异味，以及二手烟对身边人的影响和危害

#### <a href="#注意事项" class="toc-anchor"></a> <span class="text-big">**注意事项：**</span>

- 香烟的剂量并没有标准的说法，只要抽了那就是有害，最安全的当然是一根不抽。一天大于10根或总吸烟时间高于30min的属于重度吸烟者，请勿尝试或长期高于这个量。
   <span class="text-tiny">（小编我自己都是微重度剂量怎么还跑来劝别人来了）</span>
- 大麻和一些其他制品的外观和卷烟基本类似，在使用来路不明的卷烟制品前请注意辨别，避免造成不必要的麻烦。
- 吸烟产生的作用主要通过血液循环实现，洗热水澡或运动后等前提下吸烟，均会促进烟草中一些物质的吸收率。
- 细支香烟的剂量约是粗支香烟的一半，虽然一次摄入的量更少了，但可能导致一次所吸的根数增加。
- 一次性香烟滤嘴可以一定程度上起到减少有害物质的作用，但同时也会对应的减少快感。
- 此外和其他药物一样，每个人的体质差异不同，额定的量也会相应不同。若感觉不适请立刻戒烟或减量。
- 香烟所含的尼古丁，其危害和成瘾性比想象中高出很多。不要以为“抽烟的人这么多肯定危害小”。实际上其本身的危害完全不容小觑，下文会着重讲述。

# <a href="#尼古丁nicotine" class="toc-anchor"></a> 尼古丁<span class="text-small">（Nicotine）</span>

尼古丁，别称烟碱，本身存在于茄科植物的细胞液中，是茄科植物特有的生物碱。燃烧后进入人体内被吸收。主要作用在人体的神经系统，是一种神经毒素。能引起兴奋，欣快感，同时也极易成瘾。

某种角度来讲，尼古丁和毒品有很多相似之处，有文献表明其成瘾性甚至高于大麻（但别拿这个当成你抽大麻的理由），或许尼古丁及香烟更应该被看待为毒品的低配版，即使在便利店可以轻松买到，但其本身的危害决对不容小觑。香烟所代表的也并非简单的消遣娱乐，而是实打实的一种影响人类健康的产物。

<span class="text-huge">**药理学：**</span>

## <a href="#基本信息" class="toc-anchor"></a> <span class="text-small">**基本信息：**</span>

尼古丁进入体内后经血液传播开来，约7s左右到达脑部，可通过血脑屏障。在人体内约两小时基本代谢完毕。吸烟摄入的尼古丁量受香烟本身及自身吸食方式等影响。

尼古丁能够收缩肺部末端的支气管，减少肺循环的效率，与一氧化碳与血红蛋白结合同时导致呼吸困难和缺氧。此外还能引起咳嗽和痰液的分泌。这便是吸烟造成咳嗽与咳痰的原因。

## <a href="#药理作用-1" class="toc-anchor"></a> <span class="text-small">药理作用：</span>

尼古丁作用于人体的神经系统，与烟碱型乙酰胆碱受体结合，增加了神经递质量，**使大脑多巴胺分泌增加**，产生幸福感和放松感，这便是吸烟获得快乐的本质。而后，代谢后的欣快感缺失是导致上瘾的罪魁祸首。

        **对神经系统：**

- 尼古丁会刺激交感神经，借由刺激内脏神经影响副肾髓质，释放肾上腺素。刺激副交感神经节前纤维释放乙酰胆碱，作用在烟酸乙酰胆碱接受器上，使之释放肾上腺素和去甲肾上腺素至血液中，从而产生快感。

        **对循环系统：**

- 可导致血管收缩从而血压升高，属于兴奋剂，以及阻碍血液循环。同时在肾上腺髓质结合后，可增加肾上腺素与去甲肾上腺素的含量，导致心跳加快，血压升高，类似于高血压的部分症状。
- 四肢血液循环也容易受其影响，增加凝血和血栓的风险。

        **对生殖系统：**

- 血管收缩后有碍于阴茎海绵体的充血，导致男性在短期内难以勃起，长期则会阳痿或性功能障碍

**        此外：**

- 尼古丁可以促使一些细胞分解的其他激素会促使脂肪分解，加上本身减少食欲的作用，能够达到一定减肥效果，使身体消瘦。

 

### <a href="#代谢-1" class="toc-anchor"></a> 代谢：

尼古丁主要由肝脏的代谢酶：细胞色素P450进行代谢，产物为**可替宁**（Cotinine）。可替宁在血液中会存在48h，可用于检测近期是否吸烟。

### <a href="#毒理学" class="toc-anchor"></a> 毒理学

尼古丁是强致命的毒物。

对大鼠的半数致死量为50 mg/kg，对小鼠的半数致死量为3
mg/kg，对成年人的半数致死量为0.5-1 mg/kg。

尼古丁目前并没有确认为致癌物，也并不被认为是导致胎儿畸形等部分病症的罪魁祸首。

## <a href="#依赖性" class="toc-anchor"></a> <span class="text-small">依赖性</span>

前面有提到，尼古丁作用于脑部后产生多巴胺，从而导致成瘾。实际上，<span class="mark pen-green">**尼古丁的成瘾性远高于大麻在内的多数毒品，仅次于古柯碱和海洛因，但是并不能成为你吸毒的理由！**</span>同时尼古丁本身还会使其对应受体减少，长期会让快感从而减少，使人进一步提高摄入量以对其加以补偿。

# <a href="#戒烟" class="toc-anchor"></a> 戒烟

香烟及尼古丁所形成的瘾本身较难戒断，不过在我国戒烟人数远高于戒毒人数的信息下，让戒烟显得反而没那么难。香烟所带来的瘾可分为心理上的和生理上的，在此我们分开来讨论。

## <a href="#生理戒烟" class="toc-anchor"></a> **生理戒烟**

吸烟本身带来的多巴胺缺失是烟瘾的罪魁祸首，虽说一些方面可以和大麻比肩，但也鲜有“不抽我就混森蓝受”，“难受的在地上打滚”，“身上有蚂蚁在爬”之类毒品一般的戒断反应。烟瘾来时试试其他方式代替，如运动，让自己以为工作之类或者就算闲事儿忙起来，吃点喝点好的，都能一定程度缓解。（此外od的确能减少烟瘾，极端情况如果本身有od的习惯可以一试，或者当做自己戒烟的开始。小编认为o一两次换来不抽烟也算不亏，啊这波咱们不亏啊，反正不戒烟时不时也会o的。。。？）。

***烟瘾大概能在21天左右消除，可以定一个小目标，也就，也就21天，只要坚持过去了，一切都会好的。***

## <a href="#心理戒烟" class="toc-anchor"></a> **心理戒烟**

你是否有诸如早起一支烟，饭后一支烟，事后一支烟，睡前一支烟，做特定事必抽烟此类的习惯？这些习惯进行久了就会导致特定时间段不抽烟比其他时候更难受。其实这样只不过是在你的日程表中形成了特定的顺序和习惯，介入改变自己的日程表可能会让这些习惯被冲散或替代，例如逼着自己做些看似很忙能骗过自己的事，饭后一支烟改为一杯咖啡牛奶或可乐，等。这样转移注意力的方式的确能起到作用。

# <a href="#编者杂谈" class="toc-anchor"></a> <span class="text-tiny">* 编者杂谈*</span>

       
在我国，香烟本来就有着较长的历史和世界最大的市场，即使一方面本身危害不言而喻，另一方面人们呼声再高，也不可能像毒品一样被加以大力度被打压。其一是作为国家垄断行业本身能带来不小的税收，其二是群体人数极高难以立刻严加管控，其三是本身在人们心中的危害也被渲染的并不高，突然说这玩意极其有害和突然说病毒没害处一样，给人造成的转折和震撼也不小。

       
同时，我们作为历史上的确被毒品严重侵害过的名族，对毒品以及药物滥用也就是od的反感，避免，打压却是位居世界前列。但对有害物品却并没有在不掺杂任何主观情感下一碗水端平的公平看待，对烟草的打压力度虽然的确已经很高了，（例如垄断，提高价格，限制种植，仅限持证销售，仅限线下销售，等）但实际上也只是微乎其微。有害性上和毒品并肩而论的东西在我国却和酒、槟榔等被一致看待甚至同样摆在便利店里，就像是值日班长喜欢上课讲话所以自习课不管纪律一样。即使行动上不严打，但香烟和尼古丁的危害在我国尤其应该被普及宣传，让人们正确且全面的去了解和认识，而非被表述的很模糊，不全面。即使不实打实的去封禁，也得让人们知道自己天天抽的到底是什么。
