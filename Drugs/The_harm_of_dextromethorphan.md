## <a href="#综述" class="toc-anchor"></a> 综述

话先先说到前面。

- **右美沙芬的生理杀伤力不高，但心理杀伤力极高，心理成瘾性极高**
- **右美沙芬可侵袭、破坏、篡改认知，让人认知不到自己处于滥用与成瘾之中，并误认为滥用与成瘾才是正常**

于<a href="/Sub-pandect_to_Overdose"
class="is-internal-link is-valid-page">Overdose分论</a>中，定性的右美沙芬副作用为：

- 人格解体与解离症，对感知具有屏蔽、麻痹、削弱能力，对认知与判断具有侵袭、破坏、篡改能力，诱发记忆障碍、思维抑制、呼吸抑制
- SSRIs撤药反应，诱发忧郁、沮丧
- 呕吐、胃溃疡，造成窒息风险

## <a href="#生理杀伤性" class="toc-anchor"></a> 生理杀伤性

尽管很多老一辈、记者、社会工作者、医生都会觉得，但右美沙芬的生理杀伤性并不出色，事实就是如此。造成这种覆盖性错误的原因，是因为许多家常药对肝肾损伤较大，但右美沙芬并不如此。我收集了很多报告，很多长期od的人员，肝肾功能都很齐全。可能开始会有些问题，但终究没有差别。至于对神经系统与大脑的损伤，实际上是相当可逆的，若不是常年累计，基本都能救

一般地说，右美沙芬对肝损伤不高，且小于酒，对肾损伤低于对肝损伤，且小于各种中药

## <a href="#心理杀伤性-成瘾性" class="toc-anchor"></a> 心理杀伤性、成瘾性

据观测，<a href="/Effects/Dissociation"
class="is-internal-link is-valid-page">解离剂</a>们不仅可以切断感知，俗称“灵魂出窍”，对认知也具有改变的。通过如下真实案例便可一探究竟：

> oder去服用超剂量的止咳药，路人觉得这是有问题的，药物怎么能乱吃然而oder却觉得毛问题都没。先不管正确与否，至少出现了观念差

> <a href="/Report/RP-49"
> class="is-internal-link is-valid-page">桜崎彩加</a>。尤其是其死前一季的时间内，笔者一直在告诉他od的威胁所在，告诉他那解离已经把他认知破坏了，他应该尽早逃离西格玛高原，可他已经被解离篡改了认知了。尽管他自己也承诺：`“我下定决心从现在开始起好好服用抗抑郁药物，绝不再od了，我太惭愧太惭愧了，我简直他妈就是个出生”`，但他三番五次地后悔：`“在此之前我情绪精神很差连续od几天了，我害怕是身体撑不住了回光返照”`。最终，在解离剂与精神分裂剂的加持下，他坠楼身亡

> 笔者的一名联络人。学会od前，她坚守自己的贞洁道德，坚持要。学会od后，也由于贫穷与懒惰，她放弃了曾经坚守的贞洁道德，卖出了性的第一次。笔者曾建议她去找一份例如服务员的工作，但她拒绝了，并成为了一个职业卖淫者

就像一个怪物<sup><a href="#fn2" id="fnref2">[2]</a></sup>，欺骗了受害者，让他误以为怪物是自己的家人，跟着一起进屋一起生活，吸食受害者的生命力……直到被害者死去也不知道，怪物不是他的家人……原来诱骗不仅存在于教唆人od的锑之中，还本身就存在于药物里！

这出现了一个最恐怖的死循环：**因为od而认知不到危害，因为od而误以为od才是正常，因为认知不到危害存在而继续od，因为od而……**

这就是右美沙芬强大的成瘾性的来源，甚至于解离还会放大一切的依赖与成瘾：**因为od而认知不到自己正在依赖药物，因为od而误以为只有od才能满足欲望，因为认知不到自己已经成瘾而继续od，因为od而……**

oder倘若不知道自己的认知已经被篡改了，那么一辈子也无法逃离西格玛高原，就像没学过英文的人一辈子也听不懂英语。实际上还有<a href="/zh/Drugs/SRIs"
class="is-internal-link is-valid-page">血清素再摄取抑制</a>共有的SSRIs撤药反应，但那原理上无法造成任何成瘾，也相当可控，不做太多论述

- **危害是损害服药者，更危害的是让服药者认知不到自己正在被损害，最危害的是让服药者误以为危害才是正常的**
- **成瘾是控制服药者，更成瘾的是让服药者认知不到自己正在被控制，最成瘾的是让服药者误以为控制才是正常的**

右美把这六条都占了。于是，我们将右美沙芬的罪行总结出来：

- **右美沙芬可侵袭、破坏、篡改认知**
- **右美沙芬引起的解离，可以侵袭心理防御，破坏并篡改心理认知**
- **右美沙芬具有强大的心理成瘾性，生理成瘾性很小**
- **右美沙芬让人认知不到自己处于滥用与成瘾之中，并误认为滥用与成瘾才是正常**

读者可以自行对照，自己曾经的观念与现在有什么不同，以防止心理杀伤力

断药后所产生的反应，说是戒断反应也行吧，反正症状就那样，怎么说都行

## <a href="#建议" class="toc-anchor"></a> 建议

关于戒断反应的一些细节建议：可将od行为替换为某些沉溺但容易脱离的行为，比如跑步引起的气喘，图画出内心不安的画面（无论多少，只要你满意就好）安全性的打击/撕扯，踢踹等行为（比如对着大型抱枕一直这样做，能够保证你的安全与情绪能够得到有效安抚）

关于一些奇妙的小报告，经调查发现，家里有小动物的od人员如果小动物待在一起时会忘记自身的od行为，所以养宠物也是能够帮助戒断与脱离自身负面情绪的不错选择
