# <a href="#阿立哌唑" class="toc-anchor"></a> 阿立哌唑

> **阿立哌唑我恨你**

**阿立哌唑(Aripiprazole)**
是非典型抗精神病药，属于多种神经递质受体拮抗剂、抗抑郁剂。**无法滥用**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

> **罄竹难书**

#### <a href="#作用机理1" class="toc-anchor"></a> 作用机理<sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 部分激动受体：血清素/5-HT<sub>1A</sub>, 5-HT<sub>2A</sub>,
  5-HT<sub>2C</sub>, 5-HT<sub>7C</sub>
- 反向激动受体：血清素/5-HT<sub>2B</sub>
- 完全激动受体：血清素/5-HT<sub>5A</sub>
- 拮抗受体：血清素/5-HT<sub>2A</sub>（功能性）,
- 拮抗受体：α<sub>1A</sub>, α<sub>1B</sub>, α<sub>2A</sub>,
  α<sub>2C</sub>
- 拮抗受体：多巴胺/D<sub>1</sub>, D<sub>2L</sub>
- 部分激动受体：多巴胺/D<sub>2</sub>, D<sub>3</sub>, D<sub>4</sub>
- 拮抗受体：组胺/H<sub>1</sub>
- 拮抗受体：NMDA
- 抑制再摄取：血清素/STN
- 抑制再摄取：正肾素/NE
- 抑制再摄取：多巴胺/DA

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：情感屏蔽<sup><a href="#fn2" id="fnref2">[2]</a></sup>**
- 主要的主作用：抗精神分裂、反幻觉、抗双向情感障碍、抗抑郁

#### <a href="#药代动力学3" class="toc-anchor"></a> 药代动力学<sup><a href="#fn3" id="fnref3">[3]</a></sup>

阿立哌唑的口服片剂生物利用度约为90%。表现出线性动力学，消除半衰期约为75小时。约14天内，达到稳态血浆浓度。口服3~5h后，血浆浓度达峰

阿立哌唑主要通过CYP2D6和CYP3A4酶进行广泛的肝代谢（脱氢、羟基化和N-脱烷基化）
。其唯一已知的活性代谢物是脱氢阿立哌唑，通常累积至阿立哌唑浓度的约40%。胃肠外药物仅微量排泄，其代谢物均通过粪便和尿液排泄。

## <a href="#药物使用" class="toc-anchor"></a> 药物使用

> **这不是能od的药物，无法提供任何滥用剂量与期间的数据**

医学剂量：<sup><a href="#fn4" id="fnref4">[4]</a></sup>\
成人口服起始剂量，一次10mg或一次15mg，一日1次，用药2周内不应增加剂量，2周后根据疗效和耐受情况而缓慢增加剂量，最大可增至30mg，此后，可维持此剂量不变。有效剂量范围为一日10~30mg。

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

阿立哌唑会干扰<a href="/Drugs/DXM" class="is-internal-link is-valid-page">右美沙芬</a>的代谢
