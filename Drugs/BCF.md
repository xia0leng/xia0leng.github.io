# <a href="#巴氯芬" class="toc-anchor"></a> 巴氯芬

> 狗喂巴氯芬，越喂越牛逼！

**巴氯芬(Baclofen)**
是抗痉挛药，属于去焦虑剂、兴奋剂。一般以盐酸配合物，即**盐酸巴氯芬**。巴氯芬是<a href="/Drugs/SPM" class="is-internal-link is-valid-page">GABA</a>的衍生物，即β-(对氯苯基)-GABA

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理1" class="toc-anchor"></a> 作用机理<sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 激动受体：GABA<sub>B</sub>
- 阻断通道：Ca<sup>2+</sup>（电压依赖性α~2δ亚基）（这点与<a href="/Drugs/PR" class="is-internal-link is-valid-page">普瑞巴林</a>相似）（由此引发以下作用）
- 抑制释放：乙酰胆碱/ACh
- 抑制释放：谷氨酸/Glu
- 抑制释放：P物质/SP/Substance P
- 抑制释放：正肾素/NE
- 间接提高水平：GABA

#### <a href="#药物作用2" class="toc-anchor"></a> 药物作用<sup><a href="#fn2" id="fnref2">[2]</a></sup>

- **主要的副作用：成瘾、阿兹海默症、认知降低、焦虑、噩梦、晕迫、胃溃疡**【此项并不完善】
- **急性的副作用：记忆力衰退**
- 主要的主作用：社交能力提升
- 急性的主作用：镇静、去除焦虑、兴奋、<a href="/Effects/Dissociation"
  class="is-internal-link is-valid-page">解离</a>

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

巴氯芬口服吸收迅速，广泛分布于全身，生物利用率低。巴氯芬主要以原形通过肾脏排泄，半衰期约为2h~4h，因此需要全天频繁使用，以适当控制痉挛

## <a href="#药物使用3" class="toc-anchor"></a> 药物使用<sup><a href="#fn3" id="fnref3">[3]</a></sup>

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

我懒得写

#### <a href="#剂量以60kg10mg为1t无耐药性为标准" class="toc-anchor"></a> 剂量<sub>【以60kg，10mg为1t，无耐药性为标准】</sub>

医学剂量：0.5t~1t，每日三次\
起效剂量：0.5t\
轻度剂量：1t~2t\
中度剂量：2t~5t\
高度剂量：5t~10t\
重度剂量：10t+\
死亡剂量：12t+会有昏迷风险\
推荐剂量：3t\
最高剂量：10t

#### <a href="#期间-以距离服药已过去的时间为标准缺少数据"
class="toc-anchor"></a> 期间 <sub>【以距离服药已过去的时间为标准】【缺少数据】</sub>

起效时：30min~75min\
来效时：2h~3h\
顶峰时：1h~1h30min\
残效时：6h~12h\
总效时：8h~14h

> 需注意，体重不同剂量不同，上述剂量是针对60kg的人的。你的体重除以60，再乘以上述之中的剂量，即为适应你的剂量

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

巴氯芬常被用以和<a href="/Drugs/PR" class="is-internal-link is-valid-page">普瑞巴林</a>配伍
