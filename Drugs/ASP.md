# <a href="#氨磺必利" class="toc-anchor"></a> 氨磺必利

**氨磺必利(Amisulpiride)**
别名**阿米舒必利**，俗称百优林，是一种非典型抗精神病药、抗精神分裂药、止吐药。**无法滥用**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理1" class="toc-anchor"></a> 作用机理<sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 拮抗受体：α<sub>2A</sub>, α<sub>2C</sub>
- 拮抗受体：多巴胺/D<sub>2</sub>, D<sub>3</sub>, D<sub>4</sub>
- 拮抗受体：血清素/5-HT<sub>1B</sub>, 5-HT<sub>1D</sub>,
  5-HT<sub>2A</sub>, 5-HT<sub>2B</sub>, 5-HT<sub>6</sub>,
  5-HT<sub>7</sub>
- 拮抗受体：GHB

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：不适、嗜睡、乏力、震颤**
- 主要的主作用：抗精神分裂、反幻觉

#### <a href="#药代动力学2" class="toc-anchor"></a> 药代动力学<sup><a href="#fn2" id="fnref2">[2]</a></sup>

氨磺必利可从胃肠道吸收，生物利用度为43%～48%。口服后出现两次血药峰值，分别出现与1h、3~4h。蛋白结合率很低，代谢率也很低，大量原药随尿排出。终末半衰期约为12h。
