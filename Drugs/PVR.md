# <a href="#喷托维林" class="toc-anchor"></a> 喷托维林

> 喷托维林基本不能用于od，且作为谵妄剂具有高毒性

**喷托维林(Pentoxyverine)**
是中枢神经止咳药，属于谵妄剂，为复方药物。一般以枸橼jǔ
yuán酸配合物，即**枸橼酸喷托维林**。**基本不能od**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理-1" class="toc-anchor"></a> 作用机理 <sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 激动受体：σ-1/Sigma-1
- 拮抗受体：毒蕈碱乙酰胆碱/mAChR/M<sub>1</sub>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：<a href="/Effects/Delirium"
  class="is-internal-link is-valid-page">谵妄</a>、抗胆碱能综合征、精神病状、呼吸抑制**
- 主要的主作用：镇静

#### <a href="#药代动力学-2" class="toc-anchor"></a> 药代动力学 <sup><a href="#fn2" id="fnref2">[2]</a></sup>

喷托维林以肠道快速吸收，约2h，血药浓度达峰，若直肠给药则需约4h。由于首过效应超过50%，栓剂的生物利用度约为口服制剂的两倍。最重要的代谢是酯水解

于半衰期，口服制剂为2.3h，栓剂为3h~3.5h

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

喷托维林的止咳原理等同于右美沙芬：激动Sigma-1受体

由于抗胆碱功力，喷托维林不适合od
