# <a href="#地芬诺酯" class="toc-anchor"></a> 地芬诺酯

> 地芬诺酯不是<a href="/Drugs/DPD" class="is-internal-link is-valid-page">地芬尼多</a>，还隐含了<a href="/Drugs/Opioids"
> class="is-internal-link is-valid-page">阿片类成瘾</a>，甚至被迫<a href="/Drugs/compound"
> class="is-internal-link is-valid-page">复方</a>了抗胆碱药物阿托品，使用体验还不如<a href="/Drugs/EtOH" class="is-internal-link is-valid-page">酒</a>或<a href="/Drugs/Cigarette"
> class="is-internal-link is-valid-page">香烟</a>

**地芬诺酯(Diphenoxylate)**
是抗腹泻药、抗放射病药，用于急慢性功能性腹泻、慢性肠炎，属于阿片类药物<sup><a href="#fn1" id="fnref1">[1]</a></sup>。由于其搭配阿托品可更好止泻，且为了防止滥用，法律上一律只允许以阿托品配合物（阿托品还是生物碱），即**复方地芬诺酯**。地芬诺酯是哌替啶的衍生物

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理方便起见包括地芬诺辛" class="toc-anchor"></a> 作用机理<sub>（方便起见，包括地芬诺辛）</sub>

- <a href="/zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类麻醉</a>（低剂量时针对肠道系统，不涉及精神作用。高剂量时丢失针对性，开始涉及精神作用）

#### <a href="#作用2" class="toc-anchor"></a> 作用<sup><a href="#fn2" id="fnref2">[2]</a></sup>

- **主要的副作用：腹胀、肠梗阻、记忆力废退、注意力涣散、严重嗜睡、<a href="/zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类成瘾</a>**
- **急性的副作用：便秘、呼吸抑制、昏迷昏睡、恶心、<a href="/zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类中毒</a>**
- 主要的主作用：止泻、（大剂量下）<a href="/zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类麻醉</a>

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

地芬诺酯口服吸收迅速，2小时，血药浓度达峰。主要代谢物为地芬诺辛，止泻作用比地芬诺酯强5倍

半衰期上，地芬诺酯为2.5小时，地芬诺辛为12

## <a href="#药物使用" class="toc-anchor"></a> 药物使用

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

懒得写品牌

复方地芬诺酯片：\
每片含2.5mg盐酸地芬诺酯、(25μg=0.025mg)硫酸阿托品。<sup><a href="#fn3" id="fnref3">[3]</a></sup>

也只有复方。单方地芬诺酯一定是非法的<sup><a href="#fn4" id="fnref4">[4]</a></sup>

#### <a href="#剂量-以60kg无耐药性复方地芬诺酯为标准5"
class="toc-anchor"></a> 剂量 【以60kg，无耐药性，复方地芬诺酯为标准】<sup><a href="#fn5" id="fnref5">[5]</a></sup>

> 地芬诺酯玩毛线玩，吃多了拉都拉不出来。笔者推荐你吃<a href="https://mcdonalds.com.cn/" class="is-external-link"
> rel="noreferrer" target="_blank">这个</a>或者<a href="/zh/Drugs/DXM" class="is-internal-link is-valid-page">这个</a>

医学剂量：口服，每次2.5mg ~ 5mg，2次 ~
4次每天。至腹泻被控制时，应即减少剂量

200mg/kg为人最小致死量，即12g

大剂量（每次40～60mg）可产生欣快感

**拒绝写入滥用剂量**

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

地芬诺酯是阿片类里最上不去下不来的了，上不去是因为复方了阿托品，体验肠梗阻很难受，下不来是因为好歹是阿片类能被盯上列管。这种东西没有滥用的价值，就像甲苯一样
