# <a href="#奈福泮" class="toc-anchor"></a> 奈福泮

> **奈福泮的精神药效尚且不确定且危险性较高，请不要贸然尝试。试验人员正在测试，稍安勿躁**

> **不要靠近了，奈福泮会导致严重的高血压，可当场击杀服药者**\
> 奈福泮必须搭配降压药服用 并实时监测血压心率 发生不适时请紧急就医\
> 患有心脑血管者禁止服用奈福泮

**奈福泮(Nefopam)**
是非麻醉性镇痛药（手术后、癌痛、急性外伤痛等），属于抗抑郁剂。一般以盐酸配合物，即**盐酸奈福泮**。药片味道为无味的盐酸，令人感到清凉

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理-1" class="toc-anchor"></a> 作用机理 <sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 抑制再摄取：<a href="/Drugs/SRIs" class="is-internal-link is-valid-page">血清素</a>、正肾素、多巴胺（三重单胺再摄取抑制）
- 激动受体：血清素/5-HT<sub>2A</sub>, 5-HT<sub>2B</sub>,
  5-HT<sub>2C</sub>
- 阻断通道：Na<sup>+</sup>, Ca<sup>2+</sup>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：高血压、尿血、立竿见影的肾损伤、记忆力废退、抑郁、心动过速、精神分裂**【缺少数据】
- 主要的主作用：抗抑郁、镇痛、欣快、外部幻觉

## <a href="#药代动力学2" class="toc-anchor"></a> 药代动力学<sup><a href="#fn2" id="fnref2">[2]</a></sup>

奈福泮的绝对生物利用度较低，约0.5h ~ 2h起效，1h ~
3h血药达峰，于肝脏通过N-去甲基化等途径代谢，主要以尿液消除，少量以粪便

于终末半衰期，奈福泮为3h ~ 8h，其活性代谢物去甲基奈福泮为10h ~
15h且不具有精神活性

## <a href="#一些事项3" class="toc-anchor"></a> 一些事项<sup><a href="#fn3" id="fnref3">[3]</a></sup>

奈福泮可以降低胆碱水平

于镇痛作用，20mg奈福泮约等价于12mg吗啡

奈福泮带来的心跳加快源自于过多NE激动β1受体

NE主要激动α受体 微弱激动β1受体 α受体被激动引发血管收缩 加上心跳的加快
让人体进入战逃反应 出现高血压的症状

旅途中建议配合降压药使用 并实时监测血压心率 发生不适时请紧急就医

阻断Na<sup>+</sup>,
Ca<sup>2+</sup>通道可以抑制谷氨酸传递，这点相似于<a href="/Effects/Dissociation"
class="is-internal-link is-valid-page">解离</a>
