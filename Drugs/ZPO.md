# <a href="#扎来普隆" class="toc-anchor"></a> 扎来普隆

**扎来普隆(Zaleplon)**
是非苯二氮卓类的新型安眠药、抗入睡困难药，属于镇静剂与内部幻觉剂。引用自『NEEDY
GRIL OVERDOSE』的药物「海波隆 CP」，还可以称其为**海波隆**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理" class="toc-anchor"></a> 作用机理

- 激动受体：GABA<sub>A</sub>（α亚单位的ω-1受体）
- 正构受体：GABA<sub>A</sub>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：记忆削弱、遗忘**
- 主要的主作用：镇静、入眠强化、多梦

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

**扎来普隆的半衰期只有1h**。扎来普隆主要经过CYP3A4代谢，约9%的剂量被代谢。45min以内，血药浓度达峰

## <a href="#药物使用" class="toc-anchor"></a> 药物使用

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

【缺少数据】

#### <a href="#剂量-以60kg无耐药性为标准1" class="toc-anchor"></a> 剂量 <sub>【以60kg，无耐药性为标准】</sub><sup><a href="#fn1" id="fnref1">[1]</a></sup>

医学剂量：5mg~10mg一次，每日睡前一次

滥用剂量不明

#### <a href="#期间-以距离服药已过去的时间为标准2" class="toc-anchor"></a> 期间 <sub>【以距离服药已过去的时间为标准】</sub><sup><a href="#fn2" id="fnref2">[2]</a></sup>

【缺少数据】

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

扎来普隆的半衰期仅仅只有1h，于此再加上睡眠八小时后，扎来普隆就完全排出了。如此，药物副作用也会减少很多

扎来普隆更适用于入睡困难
