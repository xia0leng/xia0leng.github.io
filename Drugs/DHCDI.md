# <a href="#二氢可待因白兔bron其一" class="toc-anchor"></a> 二氢可待因/白兔BRON其一

> 二氢可待因为阿片类药物，具有最恐怖的<a href="../zh/Drugs/Opioids"
> class="is-internal-link is-valid-page">阿片类成瘾</a>。尽管二氢可待因的作用程度较低，成瘾效率不大，但仍请警惕

关于杂项与报告，请看<a href="DHCDI/OAR"
class="is-internal-link is-valid-page">二氢可待因-杂项与报告</a>

**二氢可待因(Dihydrocodeine, DHCDI)**
是成瘾性中枢止咳药、止痛药，属于阿片类麻醉剂。更多地以磷酸配合物，即**磷酸二氢可待因**。化学上可称作**3-甲基二氢吗啡**，也就是说，二氢可待因是二氢吗啡的前药<sup><a href="#fn1" id="fnref1">[1]</a></sup>

#### <a href="#作用机理" class="toc-anchor"></a> 作用机理

  class="is-internal-link is-valid-page">阿片类麻醉</a>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：<a href="../zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类成瘾</a>、记忆力废退、注意力涣散、严重嗜睡**
- **急性的副作用：<a href="../zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类中毒</a>、便秘、呼吸抑制、昏迷昏睡、恶心**
- 主要的主作用：<a href="../zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类麻醉</a>、幸福感、梦境增强、欣快感

> **关于阿片类成瘾的危害与杀伤力，请看<a href="../zh/Drugs/Opioids"
> class="is-internal-link is-valid-page">阿片类成瘾</a>条目**

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

二氢可待因具有与<a href="../zh/Drugs/DXM"
class="is-internal-link is-valid-page">右美沙芬</a>相似的代谢机制。二氢可待因本身药效很小，约10%经CYP2D6代谢为有效产物**二氢吗啡**，其余经CYP3A4代谢为近乎无效的产物N-去甲二氢可待因。前者就是二氢可待因药效的缘由。代谢快慢，似乎对二氢可待因最终的药效影响不大

也就是说，二氢可待因本身不怎么起作用，全靠代谢出来的二氢吗啡起作用。因此可以说，二氢可待因是10%的吗啡，二氢可待因等价<a href="./Drugs/MOP" class="is-internal-link is-valid-page">吗啡</a>

抑制CYP3A4的西柚汁，同样也能增强二氢可待因的药效，因为于此，有更多的二氢可待因转化为了二氢吗啡。**但抑制CYP2D6的<a href="./Drugs/SRIs" class="is-internal-link is-valid-page">SRIs</a>等药物，则可能完全抑制二氢可待因转化为二氢吗啡，使药效无效化。这可能是有些人员od二氢可待因没有药效的原因**

二氢可待因的半衰期约为4小时<sup><a href="#fn2" id="fnref2">[2]</a></sup>。约1小时，血药浓度达到峰值

## <a href="#白兔bron" class="toc-anchor"></a> 白兔BRON

> 白兔EVE不是白兔BRON，白兔EVE完全没有滥用潜力

白兔BRON是由日本SS制药株式公社，生产的止咳药。白兔为香港商标名，BRON是药物商品名。<sup><a href="#fn3" id="fnref3">[3]</a></sup>白兔BRON因『主播女孩重度依赖』而闻名于od社区，具有精致的外观。玻璃瓶与金属瓶盖的白兔BRON，收藏价值很高

实际上，在NGOD里所描述白兔BRON的药效，是有偏差的。二氢可待因并不是很能引起那样五颜六色的视觉幻觉（<a href="../zh/Drugs/DHCDI/OAR"
class="is-internal-link is-valid-page">见此处</a>）。而短片<a href="https://www.bilibili.com/video/BV1oL41187aJ"
class="is-external-link" rel="noreferrer" target="_blank">梦充夜</a>中所描述的，在内部幻觉中见到超天酱降临的药效，便正确地描述了二氢可待因的内部幻觉

白兔BRON一般选用瓶装，有84片、60片两种规格，每片成分见<a href="./Drugs/compound"
class="is-internal-link is-valid-page">复方专题</a><sup><a href="#fn4" id="fnref4">[4]</a></sup>

42片白兔BRON，就等于105mg二氢可待因<sup><a href="#fn5" id="fnref5">[5]</a></sup>，效力稍大于10mg吗啡，而吗啡的一般用量就是10mg。况且还有315mg咖啡因，与也不知道能干啥的175mg甲麻黄碱。

因此，对于60kg的人员来说，每次应该od，半瓶或一瓶白兔BRON，即42片或84片。若剂量不够，则可能毫无药效（有报告佐证）。

> 需注意，体重不同剂量不同，上述剂量是针对60kg的人的。你的体重除以60，再乘以上述之中的剂量，即为适应你的剂量。

## <a href="#药物使用" class="toc-anchor"></a> 药物使用

#### <a href="#药物品牌" class="toc-anchor"></a> 药物品牌

复方制剂——

- 氨酚双氢可待因片：盒装2.5mg×每板12片×2板，价格不明\
  　　每片成分：扑热息痛500mg、酒石酸二氢可待因10mg

- **白兔BRON**：瓶装2.5mg×每瓶84或60片，价格约为84元到120元\
  　　每片成分：磷酸二氢可待因2.5mg、盐酸甲麻黄碱4.2mg、马来酸氯苯那敏0.7mg、无水咖啡因7.5mg<sup><a href="#fn6" id="fnref6">[6]</a></sup>

#### <a href="#剂量-以60kg无耐药性无联合药物为标准7" class="toc-anchor"></a> 剂量 <sub>【以60kg，无耐药性，无联合药物为标准】</sub><sup><a href="#fn7" id="fnref7">[7]</a></sup>

由于二氢可待因近似于10%的吗啡，故二氢可待因的剂量，略小于吗啡的1/10。

医学剂量：基本上是一片到两片，每日三四次\
起效剂量：30mg~50mg\
轻度剂量：50mg~100mg\
中度剂量：100mg~150mg\
高度剂量：150mg~200mg\
重度剂量：200mg以上\
致死剂量：200mg以上。已知2000mg至少是致死剂量\
推荐剂量：80mg或100mg\
最高剂量：200mg

**期间：**
<sub>【以距离服药已过去的时间为标准】</sub>【缺少数据】<sup><a href="#fn8" id="fnref8">[8]</a></sup>\
起效时：15min~45min\
顶峰时：1h~1.5h\
残效时：24h\
总效时：3h~6h

> 需注意，体重不同剂量不同，上述剂量是针对60kg的人的。你的体重除以60，再乘以上述之中的剂量，即为适应你的剂量。
