# <a href="#槟榔碱" class="toc-anchor"></a> 槟榔碱

> 槟榔不是毒品，槟榔是毒药

**槟榔碱(Arecoline)**
是槟榔的主要活性成分、食品药品添加剂，属于兴奋剂。其精神作用与尼古丁相当。<sup><a href="#fn1" id="fnref1">[1]</a></sup>槟榔碱是2B级致癌物质，且具有各种的毒性。不过呢，槟榔碱是合法的

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理" class="toc-anchor"></a> 作用机理

- 部分激动：毒蔁碱乙酰胆碱/mAChR

#### <a href="#作用毒理2" class="toc-anchor"></a> 作用毒理<sup><a href="#fn2" id="fnref2">[2]</a></sup>

- 细胞毒性：肝细胞、骨骼细胞、淋巴细胞、神经元细胞、成肌细胞、内皮细胞
- 导致仓鼠：卵巢染色体畸变，抑制细胞DNA修复
- 导致小鼠：精液粘稠并失去活性、睾丸出现巨大空泡，曲细精管无法产生精子

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- 主要的副作用：器官受损、性功能受损、毒性、成瘾
- 主要的主作用：兴奋、欣快感、镇痛

#### <a href="#药代动力学3" class="toc-anchor"></a> 药代动力学<sup><a href="#fn3" id="fnref3">[3]</a></sup>

槟榔碱通过肾脏和肝脏代谢。目前已有11种槟榔碱代谢物被记录。据说石灰可以将几乎所有槟榔碱水解成槟榔啶，一种GABA再摄取抑制剂。槟榔碱和槟榔啶的主要代谢物是N-甲基哌啶酸

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

不值得为了一点激动胆碱受体而被毒

槟榔对脸部的影响极大，例如严重损害口腔健康之类，会严重损伤牙龈和脸部最后嘴张不开，然后割脸啥的。
