# <a href="#美金刚" class="toc-anchor"></a> 美金刚

> 灾难与毁灭的傻逼药物，兼全od药物中副作用最大的药物

> 由于致**重症抑郁**、精神分裂、人格解体效力，严禁**抑郁症**、精神分裂症、人格解体症患者使用美金刚——<a href="/Report/RP-50" class="is-internal-link is-valid-page">RP-50</a>

**美金刚(Memantine, MMT)**
是抗重度阿兹海默症药，属于解离麻醉剂与非血清素能致幻剂与精神分裂剂。一般以盐酸配合物，即**盐酸美金刚**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理" class="toc-anchor"></a> 作用机理

- 激动受体：多巴胺D<sub>2</sub>受体/D<sub>2</sub>
- 拮抗受体：5-HT<sub>3</sub>（非竞争性）
- 拮抗受体：NMDA（非竞争性，电压依赖性）
- 抑制受体：烟碱乙酰胆碱/nAChR
- 激动受体：σ-1/Sigma-1

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：精神分裂、大脑空洞、记忆力废退、认知丧失、阿兹海默症老年痴呆、抑郁、胃溃疡**
- **急性的副作用：记忆力丧失、抗胆碱能综合征、呕吐、晕迫、脱力、震颤、呆滞、呼吸抑制**
- 主要的主作用：精神分裂
- 急性的主作用：镇静、外部幻觉、<a href="/Effects/Dissociation"
  class="is-internal-link is-valid-page">解离</a>、内部幻觉

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学：

美金刚在体内代谢至少代谢成四种无效化物质，80%处于原形以肾经尿排泄。3~8小时，血药浓度达到峰值。**半衰期约为60h
~ 100h，一般取72小时，是全od药物中半衰期最高的药物**

美金刚几乎没有耐药性

## <a href="#美金刚的精神分裂" class="toc-anchor"></a> 美金刚的精神分裂

> 精神分裂非常折磨人，请不要抱着好奇的想法去尝试

【此段原纪录找不到了，为笔者对报告的记忆进行的叙述，故可能存在偏差】\
据一名对美金刚产生了极大耐药性的人员报告，在美金刚的精神分裂下，该人员：

- 躺在床上，闭着眼睛，并没睡着。
- 幻听见如同梦境的声音，并幻视见一些奇怪的。
- 一睁眼，声音还在，也好像又能看见一些东西

**这些都是精神分裂的症状。请各位在产生这些症状后立刻撤药**

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

外部幻觉与精神分裂，来源于Sigma-1受体激动，与多巴胺D2受体激动。**没有金刚烷胺的外部幻觉强烈**

内部幻觉可能来源于特殊的NMDA解离

美金刚引起的抑郁，来源于5-HT<sub>3</sub>受体拮抗，与多巴胺消耗

美金刚具有od药物中最长的半衰期，足足有**三天**！，这意味着美金刚的排除时间能长到**一个月**！，也意味着短期内大量服用美金刚，会导致极为灾难与长期的副作用

美金刚具有养护神经元的作用，但这只是因为电压依赖性的NMDA解离下，解离程度不会太大。相当于过载就关发动机来保护车子，一样地不可取

class="is-internal-link is-valid-page">右美沙芬</a>，也不要吃美金刚

美金刚引起极为严重的长期外部幻觉，或者说精神分裂，原因是在多巴胺D2受体激动，与Sigma-1受体激动下，服药者将产生精神分裂。**那些幻觉就是精神分裂的表现**。易知，美金刚的幻觉不如<a href="/zh/Drugs/ATD"
class="is-internal-link is-valid-page">金刚烷胺</a>，但精神分裂风险却远远大于金刚烷胺

美金刚引起记忆力废退，抗胆碱和解离干的

**美金刚所具有的毒性，在全od药物中，是最大的。**

金刚烷胺不能和<a href="/zh/Drugs/PR"
class="is-internal-link is-valid-page">普瑞巴林</a>联用。由此可推出，美金刚也不能和普瑞巴林联用

虽然美金刚化学结构与<a href="/zh/Drugs/ATD"
class="is-internal-link is-valid-page">金刚烷胺</a>相似（美金刚又名二甲金刚烷胺），但两者药效差距很大

简单的定理：药效上，美金刚 = <a href="/zh/Drugs/DXM"
