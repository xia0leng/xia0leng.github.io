# <a href="#吗啡" class="toc-anchor"></a> 吗啡

> 别再靠近了！吗啡具有最恐怖的<a href="/zh/Drugs/Opioids"
> class="is-internal-link is-valid-page">阿片类成瘾</a>，并已经可以算作非法的毒品了，请谨慎对待！

**吗啡(Morphine, MOP)**
是镇痛药（多用于重度创伤、战伤、烧伤、癌痛），属于阿片类麻醉剂。有硫酸配合物形式，更多地以盐酸配合物，即**盐酸吗啡**。吗啡是鸦片的主要活性成分

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理" class="toc-anchor"></a> 作用机理

- <a href="/zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类麻醉</a>

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：<a href="/zh/Drugs/Opioids"
  class="is-internal-link is-valid-page">阿片类成瘾</a>、记忆力废退、注意力涣散、严重嗜睡**
- **急性的副作用：阿片类中毒、便秘、呼吸抑制、昏迷昏睡、恶心**
- 主要的主作用：阿片类麻醉、幸福感、梦境增强、欣快感

#### <a href="#药代动力学1" class="toc-anchor"></a> 药代动力学<sup><a href="#fn1" id="fnref1">[1]</a></sup>

吗啡的吸收方式较多，包括口服、注射、吸入、舌下、含服、鼻内等。吗啡的口服吸收率低，仅有40%；注射吸收率高，有60%。故临床上多以注射吗啡给药

若注射，约20min内，血药浓度达到峰值；若口服，约30min内，血药浓度达到峰值

吗啡在体内广泛代谢，可能代谢成少量的去甲吗啡、可待因和氢吗啡酮

吗啡的消除半衰期约为2h，男性和女性之间可能略有差异

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

**非医用吗啡是非法的**，属于**吸毒**，被尿检查出来就纠缠不清了。请谨慎对待吗啡

**<a href="/zh/Drugs/Opioids"
class="is-internal-link is-valid-page">阿片类成瘾</a>无解**

虽然上述所说，吗啡的药效只有6h，但吗啡的后遗症可以长达一周，后遗症包括上面提到的副作用

吗啡引起的困倦、注意力涣散，是因为阿片类麻醉

据说，<a href="/Drugs/CFI" class="is-internal-link is-valid-page">咖啡因</a>是吗啡的解毒剂

等吗啡剂量下，<a href="/Drugs/compound"
class="is-internal-link is-valid-page">复方甘草片</a>与单方吗啡制剂，药效有差异。考虑到复方的其他成分掺杂，复方甘草片的药效弱于单方吗啡制剂，且副作用增强，但复方甘草片比单方吗啡制剂要更容易获取

罂粟壳所含有的吗啡含量大约为0.1%，这意味着1g罂粟壳含有1mg吗啡。这意味着约160朵罂粟壳可以提取出10mg吗啡<sup><a href="#fn9" id="fnref9">[9]</a></sup>——你最好别这么干，我只是想说罂粟壳多了照样也有精神药效，照样会出问题

在战地1战地5中，萨尼那能一针救活人的针筒，所含的药剂成分正是吗啡，虽然战地1的那药剂可能也含有可卡因
