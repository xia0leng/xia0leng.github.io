# <a href="#异丙嗪" class="toc-anchor"></a> 异丙嗪

> 基本不能用作od

**异丙嗪(Promethazine, PMZ)**
是早年的抗过敏药、抗晕动症药，属于谵妄剂、解离剂、抗精神病抑制剂。一般以盐酸配合物，即**盐酸异丙嗪**

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理1" class="toc-anchor"></a> 作用机理<sup><a href="#fn1" id="fnref1">[1]</a></sup>

- 拮抗受体：α<sub>1A</sub>, α<sub>1B</sub>, α<sub>1D</sub>,
  α<sub>2A</sub>, α<sub>2B</sub>, α<sub>2C</sub>
- 拮抗受体：β<sub>1</sub>, β<sub>2</sub>
- 拮抗受体：多巴胺/D<sub>1</sub>, D<sub>2</sub>, D<sub>3</sub>
- 拮抗受体：血清素/5-HT<sub>1A</sub>, 5-HT<sub>2A</sub>,
  5-HT<sub>2B</sub>, 5-HT<sub>2C</sub>, 5-HT<sub>6</sub>
- 拮抗受体：组胺/H<sub>1</sub>, H<sub>2</sub>
- 拮抗受体：毒蔁碱乙酰胆碱/mAChR/M<sub>1</sub>, M<sub>2</sub>,
  M<sub>3</sub>, M<sub>4</sub>, M<sub>5</sub>
- 拮抗受体：NMDA(非竞争性)
- 拮抗受体：σ-1/Sigma-1<sup><a href="#fn2" id="fnref2">[2]</a></sup>
- 抑制再摄取：正肾素/NE
- 抑制再摄取：血清素/STN

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：抑郁、谵妄、抗胆碱能综合征、阿兹海默症、认知衰退**
- 主要的主作用：抗过敏、抗精神分裂症、反幻觉、抗躁狂、镇静、解离

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

异丙嗪是脏药，堪比<a href="/Drugs/QTP" class="is-internal-link is-valid-page">喹硫平</a>的玩意

在过去，异丙嗪常与<a href="/Drugs/CDI" class="is-internal-link is-valid-page">可待因</a>作为配伍，制成紫水
