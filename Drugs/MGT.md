# <a href="#苏糖酸镁" class="toc-anchor"></a> 苏糖酸镁

> 请留心，该物质被发掘具有药理作用于2010年，文献稀缺，请谨慎对待<sup><a href="#fn1" id="fnref1">[1]</a></sup>

**苏糖酸镁(Magnesium L-threonate, MgT)**
是食品添加剂，属于反解离剂、认知增强剂、兴奋剂。一般多讨论左旋体苏糖酸镁即L-苏糖酸镁，故未标明手性的情况下，默认苏糖酸镁指代L-苏糖酸镁。笔者提议，作为反解离剂的L-苏糖酸镁可以简称为左苏，正好与解离剂的<a href="/zh/Drugs/DXM" class="is-internal-link is-valid-page">右美</a>对应

化学上，苏糖酸镁是一种有机盐，由二单位苏糖酸离子与一单位镁离子结合而成。苏糖酸的系统命名为2,3,4-三羟丁酸，CAS:
7306-96-9
<sup><a href="#fn2" id="fnref2">[2]</a></sup>。据信苏糖酸的羧基中的羟基与羰基络合了镁离子
<sup><a href="#fn3" id="fnref3">[3]</a></sup>。白色不透明结晶粉末，相似面粉。于水中粘稠成麦芽糖状，能拉丝。气味与味道接近于墙壁的腻子粉或者面粉，但并不苦涩，尚可接受。茶水可以掩盖住苏糖酸镁的味道

## <a href="#药理作用" class="toc-anchor"></a> 药理作用

#### <a href="#作用机理缺少数据" class="toc-anchor"></a> 作用机理【缺少数据】

- 补充营养：Mg2+（苏糖酸盐更能有效穿过血脑屏障）
- 协助合成：GABA、血清素、多巴胺、正肾素、肾上腺素
- 协助治疗：ADHD
- 增强能力：神经可塑性
- 增强能力：肠道蠕动
- 上调表达：海马神经元中NMDA受体的NR2B亚基

#### <a href="#药物作用" class="toc-anchor"></a> 药物作用

- **主要的副作用：窜稀、滥用下的兴奋性毒性（完全不可逆，详见<a href="/Effects/Antidissociation"
  class="is-internal-link is-valid-page">反解离</a>）<sup><a href="#fn4" id="fnref4">[4]</a></sup>**
- 主要的主作用：认知强化、<a href="/Effects/Antidissociation"
  class="is-internal-link is-valid-page">反解离</a>、抗抑郁、助眠、入睡时间缩短、稀肠
- 急性的主作用：嗅觉味觉强化

#### <a href="#药代动力学" class="toc-anchor"></a> 药代动力学

无数据

反正笔者个人觉得肝肾负担不大

#### <a href="#用法" class="toc-anchor"></a> 用法

为防止出现<a href="/Effects/Antidissociation"
class="is-internal-link is-valid-page">反解离</a>的**兴奋性毒性阿兹海默症**，请遵守下列规则

- 倘若确认有<a href="/Effects/Dissociation"
  class="is-internal-link is-valid-page">解离剂</a>的耐药性，或患有人格解体症。**一周内不要摄入大于36g，每次不要摄入大于6g**
- 倘若确认已消除解离耐药性，或从未服用解离剂。**一周内不要摄入大于12g，每次不要摄入大于4g**
- **若察觉到异常反解离状况，标志为睡眠时间减少，请立刻断药，否则将引起完全不可逆的兴奋性毒性阿兹海默症**。详见<a href="/Effects/Antidissociation"
  class="is-internal-link is-valid-page">反解离</a>
- 仅推荐使用<a href="/Drugs/EtOH" class="is-internal-link is-valid-page">酒</a>、<a href="/zh/Drugs/DXM"
  class="is-internal-link is-valid-page">右美沙芬</a>、<a href="/Drugs/MMT" class="is-internal-link is-valid-page">美金刚</a>来反制兴奋性毒性。请撤药苏糖酸镁，并每日一片美金刚，直到睡眠时间变长恢复后，立刻撤药美金刚与苏糖酸镁

> 需注意，体重不同剂量不同，上述剂量是针对60kg的人的。你的体重除以60，再乘以上述之中的剂量，即为适应你的剂量。

#### <a href="#期间" class="toc-anchor"></a> 期间

**苏糖酸镁的药效非常延迟**，且延续较久。**初次服药的药效很小，需要累积使用以达到药效**

## <a href="#一些事项" class="toc-anchor"></a> 一些事项

关于杂项与报告，请看<a href="/Drugs/MGT/OAR"
class="is-internal-link is-valid-page">苏糖酸镁-杂项与报告</a>

苏糖酸镁可以消除<a href="/Effects/Dissociation"
class="is-internal-link is-valid-page">解离剂</a>例如<a href="/zh/Drugs/DXM"
class="is-internal-link is-valid-page">右美沙芬</a>的耐药性。实际上反解离剂们都可以

不要过量苏糖酸镁，这不仅不会令药效更好，反而可能会造成阿兹海默症

滥用下，苏糖酸镁会引起阿兹海默症。见<a href="/Effects/Antidissociation"
class="is-internal-link is-valid-page">反解离</a>

苏糖酸镁联用<a href="/zh/Drugs/CFI"
class="is-internal-link is-valid-page">咖啡因</a>可以增强兴奋

苏糖酸镁本身是反解离剂，可以反制解离剂的解离；由于认知强化的药效，苏糖酸镁也可以用以反制一些，由于认知下降导致的心理疾病

其引起抗抑郁，来源于血清素、多巴胺合成协助

其引起反ADHD，来源于多巴胺合成协助

其引起反解离，来源于海马神经元中NMDA受体的NR2B亚基

其引起助眠，来源于GABA合成协助
